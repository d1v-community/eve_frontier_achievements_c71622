# Chronicle I18n Glossary

This glossary defines the canonical wording for the current `Chronicle + Warrior Share` localization pass.

## Scope Rules

- Keep product and system names in English: `Frontier Chronicle`, `Chronicle`, `Eve Eyes`, `Sui`, `SSU`, `Smart Assembly`.
- Keep each medal `subtitle` as the English canonical name.
- Localize `title`, `requirement`, `teaser`, `lore`, and all user-facing UI labels for `en`, `zh-CN`, and `is`.
- Within the Chronicle and Share surfaces, English UI copy is only allowed for the brand and system names listed above.
